User guide of using V/C

4. Use the video conference

4a. Windows:

Select the “Keypad” button, then dial the Conference Room number given by the conference admin (e.g. 3003).

You may be requested to enter the room password, it can’t be typed directly via keyboard, instead click the “Keypad” button and input it. Press # to confirm.

Done. Now you are ready to use the video conference.

4b. Mobile (Android & iOS)

Select the “Keypad” button, then dial the Conference Room number given by the conference admin (e.g. 3003).

You may be requested to enter the room password, it can’t be typed directly via keyboard, instead click the “Keypad” button and input it. Press #  to confirm.

Done. Now you are ready to use the video conference.

1. Site list

Site list shows the list of current participants in the conference room.

It also provides Chair role function, click the chairman icon (the 3rd one).

The chair role user can 1. Lock the conference  2. Invite new participant  3. Batch muting multiple users’ microphone, and more.

The chair role password is managed by the Conference Admin.

5b. I can’t attend the conference room as being kicked out

The more participants sharing their camera view, the more HKO network bandwidth is consumed. If you encounter this problem, please contact the Conference Admin team, we will try adjusting the conference quality to lower down the network bandwidth occupancy rate.

Etiquette / reminder to users for better v/c experience

6a. Video conferencing requires a stable network, so LAN cable network connection is preferred over WiFi.

6b. It’s good practice to mute the microphone when not in need of using it, this can help reduce echo noise to all participants.
